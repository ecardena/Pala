# Pala Arduino Library

Librería para mover una pala por medio de un ServoMotor en dos posiciones arriba(1) y abajo(0).

## Instalación
1. Descarga este repositorio como archivo .ZIP.
2. En el IDE de Arduino, ve a Programa > Incluir Librería > Añadir librería .ZIP.

## Uso básico
Instancia el objeto pasando el pin correspondiente, en el Setup inicia el servo con iniciaPala() y para moverlo
pala.muevePala(0) ó palamuevePala(1).
